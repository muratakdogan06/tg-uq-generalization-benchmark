# Data (MD_properties.csv)

## What this file is
`MD_properties.csv` contains the curated polymer dataset used in this study, including repeat-unit SMILES and MD-derived target/property columns (e.g., **T_g (K)**).

## Provenance (source)
The underlying molecular property data originate from Project Elwood (Materials Data Facility):

Schneider, L.; S. M.; Mysona, J.; Liang, H.; Han, M.; Rauscher, P.; Ting, J.; Venkatram, S.; Ross, R.; Schmidt, K.; Blaiszik, B.; Foster, I.; de Pablo, J.  
**Project Elwood: MD Simulated Monomer Properties.** *Materials Data Facility*, 2022.  
DOI: **10.18126/8p6m-e135**

## Notes
This repository uses repeat-unit SMILES as the structural representation for benchmarking; property labels are sourced from the dataset record above.
